Behavioral Delta Risk: Calibrated Attrition Scoring for Targeted Retention

This repository contains the code data and artifacts for a reproducible project that evaluates whether short term behavioral changes improve prediction of customer attrition and whether a tuned calibrated ensemble can support a pilot retention campaign.

The project addresses a practical business problem. The goal is to identify customers at elevated risk of attrition so that targeted outreach can be prioritized to maximize retained lifetime value. The analysis uses the anonymized BankChurners dataset provided for the capstone. A deterministic leakage audit preserves an anonymized PseudoID and removes direct identifiers and any model derived columns that could leak future information.

The research question is simple. Do engineered behavioral delta features that capture quarter to quarter changes in transaction amount and transaction count and reductions in contact frequency add predictive value beyond static demographic and credit features?

The hypothesis is that recent negative behavioral changes are associated with higher attrition risk. I predict that adding delta features to a static feature set will increase discrimination measured by AUC and will reduce the Brier score indicating better probability accuracy. I also predict that tuned ensembles will outperform logistic baselines and that Platt scaling will modestly improve calibration for the selected model.

The pipeline is fully reproducible. Data ingestion and a targeted leakage audit are followed by feature engineering median imputation rare level consolidation one hot encoding zero variance removal and optional winsorization. Modeling uses a stratified 70 30 train test split. Hyperparameter tuning uses fivefold cross validation repeated five times optimizing two class ROC. Candidate models include a static logistic an enhanced logistic a tuned random forest and an XGBoost workflow that falls back to direct xgboost cross validation when caret xgbTree produces missing ROC metrics. Final evaluation reports holdout AUC Brier and calibration diagnostics.

Status update. The preprocessing recipe and feature engineering are implemented and saved as a reproducible recipe. Behavioral delta features were engineered and validated. Random forest tuning completed successfully and produced stable cross validated results. Caret xgbTree produced missing ROC metrics in this environment so a robust direct xgboost cv fallback was implemented and validated. Models were evaluated on a held out test set and Platt scaling was applied to the selected model. Ablation experiments and bootstrap confidence intervals were computed to quantify the incremental value of behavioral features. Minimal artifacts including probability vectors ROC objects calibration outputs and bootstrap tables were exported to an artifacts folder for downstream reporting.

Outcome summary. The enhanced logistic improved AUC and Brier relative to the static logistic in the current run. Tuned ensembles substantially outperformed logistic baselines on the holdout. The direct xgboost final model produced the highest AUC and the lowest Brier in this run. Platt scaling produced a small improvement in Brier while preserving AUC. Bootstrap ablation indicates the Random Forest AUC delta from behavioral features is statistically robust and XGBoost shows a smaller positive contribution.
