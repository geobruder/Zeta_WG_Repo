Executive summary

This report documents hyperparameter tuning strategy, final tuned hyperparameters, conformal intervals and calibration diagnostics, and recommendations for further tuning and deployment.

Tuning methodology (recap)

  • Nested cross‑validation: outer 5‑fold CV for performance estimation; inner 3‑fold CV for hyperparameter search (RandomizedSearchCV).
  • Randomized search balances exploration and compute; parameter validation prevents invalid parameter choices.
  • Final refit: best hyperparameters re-searched on full cleaned dataset and final models saved.

Conformal coverage diagnostic

# The notebook computes conformal_q_90 per model and saves preds with intervals; check empirical coverage
if (!is.null(ms)) {
  cov_rows <- map_dfr(names(ms)[names(ms) != "run_info"], function(nm) {
    v <- ms[[nm]]
    q90 <- v$conformal_q_90 %||% NA
    preds_path <- path(outputs_dir, paste0(nm, "_preds_full.csv"))
    if (file_exists(preds_path) && !is.na(q90)) {
      preds <- read_csv(preds_path, show_col_types = FALSE)
      covered <- mean((preds$y_true >= (preds$y_pred - q90)) & (preds$y_true <= (preds$y_pred + q90)), na.rm=TRUE)
      tibble(model = nm, conformal_q_90 = q90, empirical_coverage = round(covered,3))
    } else {
      tibble(model = nm, conformal_q_90 = q90, empirical_coverage = NA_real_)
    }
  })
  cov_rows %>% kable(caption="Conformal q_90 and empirical coverage (held-out predictions)") %>% kable_styling(full_width = FALSE)
  cat("Interpretation: empirical_coverage close to 0.90 indicates conformal interval q_90 provides near-nominal coverage.\n")
} else {
  cat("models_summary.json not found; run nested CV cell first.\n")
}
